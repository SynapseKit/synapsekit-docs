# Guides

Step-by-step walkthroughs for building real applications with SynapseKit. Every guide is fully self-contained, includes a Google Colab notebook, and follows a consistent structure: setup → steps → complete working example → troubleshooting → next steps.

## Categories

| Category | Guides | What you'll build |
|---|---|---|
| [RAG Fundamentals](./rag/) | 8 | PDF knowledge bases, streaming RAG, hybrid search, memory-augmented pipelines |
| [Advanced Retrieval](./retrieval/) | 7 | RAG Fusion, Self-RAG, GraphRAG, parent-document retriever, cross-encoder reranking |
| [Agents & Tools](./agents/) | 10 | Research assistants, SQL agents, code execution, structured outputs, streaming |
| [Multi-Agent Systems](./multi-agent/) | 5 | Crew pipelines, supervisor routing, handoff chains, A2A communication |
| [Graph Workflows](./graph/) | 8 | Linear pipelines, conditional routing, fan-out/fan-in, human-in-the-loop, checkpointing |
| [LLM Providers & Optimization](./llms/) | 5 | Provider comparison, cost routing, fallback chains, semantic caching, structured outputs |
| [Real-World Integrations](./integrations/) | 7 | GitHub PR review, Slack bots, SQL analytics, email triage, YouTube summarizer |

**Prerequisites:** Python 3.10+, `pip install synapsekit`
